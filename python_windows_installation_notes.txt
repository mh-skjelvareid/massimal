Download Minicoda from 
https://docs.conda.io/en/latest/miniconda.html

Verify downloaded file by running the following in terminal:
CertUtil -hashfile <path_to_file> SHA256

Run conda by finding it under Anaconda in the start menu

Create environment and install packages

conda create --name massimal numpy matplotlib
conda activate massimal

pip install spectral
conda install -c conda-forge jupyterlab
conda install -c anaconda glob2
conca install scikit-image
conda install scikit-learn
conda install -c conda-forge gdal
